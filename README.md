# Online School P&L Dashboard

Интерактивный Streamlit-дашборд для расчета P&L и unit-экономики онлайн-школы на 12 месяцев.

## Что внутри
- 3 сценария агентской комиссии
- график выручки и чистой прибыли
- график окупаемости
- структура расходов
- KPI на конец 12-го месяца

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
